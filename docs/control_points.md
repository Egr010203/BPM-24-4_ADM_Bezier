# Контрольные точки

## 1. Элементарная кубическая кривая Безье для задания 3.1

Рекомендуемый пример для первой демонстрации:

| Точка | Координаты | Назначение |
|---|---:|---|
| `P0` | `(0, 0)` | начальная опорная точка |
| `P1` | `(1, 3)` | первая управляющая точка |
| `P2` | `(3, 3)` | вторая управляющая точка |
| `P3` | `(4, 0)` | конечная опорная точка |

Эти точки дают простую дугу, на которой удобно показать роль управляющих точек.

## 2. Составная кривая для задания 3.2

Выбранный контур: **стилизованное пламя**.

Контур пламени удобнее сделать не двумя, а четырьмя кубическими сегментами Безье. Так форма получается более выразительной: есть широкое основание, плавные боковые изгибы и верхняя вершина.

| Сегмент | Назначение | P0 | P1 | P2 | P3 |
|---|---|---:|---:|---:|---:|
| 1 | Нижняя левая часть пламени | `(0.0, 0.0)` | `(-2.0, -0.2)` | `(-2.8, 2.5)` | `(-1.2, 4.5)` |
| 2 | Левая верхняя часть к вершине | `(-1.2, 4.5)` | `(-1.0, 5.8)` | `(-0.5, 7.0)` | `(0.0, 8.0)` |
| 3 | Вершина и правая верхняя часть | `(0.0, 8.0)` | `(0.8, 6.8)` | `(2.0, 5.5)` | `(1.4, 3.8)` |
| 4 | Правая нижняя часть и замыкание | `(1.4, 3.8)` | `(2.8, 2.0)` | `(2.2, 0.2)` | `(0.0, 0.0)` |

## 3. Проверка стыковки

Для составной кривой важно:

```text
segment_1.P3 == segment_2.P0
segment_2.P3 == segment_3.P0
segment_3.P3 == segment_4.P0
segment_4.P3 == segment_1.P0
```

То есть:

```text
(-1.2, 4.5) == (-1.2, 4.5)
(0.0, 8.0) == (0.0, 8.0)
(1.4, 3.8) == (1.4, 3.8)
(0.0, 0.0) == (0.0, 0.0)
```

Контур получается замкнутым.

## 4. Центр масштабирования

Для контура пламени удобно использовать центр:

```text
C = (0.0, 4.0)
```

Это приблизительная середина по высоте между основанием и верхней вершиной пламени.

Масштабы для построения:

```text
k = 0.5
k = 1
k = 1.5
k = 2
```
