# Контрольные точки

## 1. Элементарная кубическая кривая Безье для задания 3.1

Рекомендуемый пример для первой демонстрации:

| Точка | Координаты | Назначение |
|---|---:|---|
| `P0` | `(0, 0)` | начальная опорная точка |
| `P1` | `(1, 3)` | первая управляющая точка |
| `P2` | `(3, 3)` | вторая управляющая точка |
| `P3` | `(4, 0)` | конечная опорная точка |

Эти точки дают простую дугу, на которой удобно показать роль управляющих точек.

## 2. Составная кривая для задания 3.2

Выбранный контур: **стилизованное пламя с тремя языками**.

Для формирования более сложного контура с тремя языками пламени используется шесть кубических сегментов Безье.

| Сегмент | Назначение | P0 | P1 | P2 | P3 |
|---|---|---:|---:|---:|---:|
| 1 | Основание -> левый язык | `(0.0, 0.0)` | `(-2.0, 1.0)` | `(-2.5, 3.0)` | `(-1.5, 4.0)` |
| 2 | Левый язык -> впадина | `(-1.5, 4.0)` | `(-1.0, 3.0)` | `(-0.8, 3.0)` | `(-0.5, 3.5)` |
| 3 | Впадина -> центр. вершина | `(-0.5, 3.5)` | `(-0.5, 5.0)` | `(-0.2, 7.0)` | `(0.2, 8.0)` |
| 4 | Центр. вершина -> впадина | `(0.2, 8.0)` | `(0.5, 6.0)` | `(0.8, 4.0)` | `(1.0, 3.5)` |
| 5 | Впадина -> правый язык | `(1.0, 3.5)` | `(1.5, 3.5)` | `(2.0, 4.5)` | `(2.2, 5.0)` |
| 6 | Правый язык -> основание | `(2.2, 5.0)` | `(2.5, 3.0)` | `(1.5, 0.5)` | `(0.0, 0.0)` |

## 3. Проверка стыковки

Для составной кривой важно обеспечить непрерывность (условие С0):

```text
segment[i].P3 == segment[i+1].P0
```

Проверка для 6 сегментов:
1. `(-1.5, 4.0) == (-1.5, 4.0)`
2. `(-0.5, 3.5) == (-0.5, 3.5)`
3. `(0.2, 8.0) == (0.2, 8.0)`
4. `(1.0, 3.5) == (1.0, 3.5)`
5. `(2.2, 5.0) == (2.2, 5.0)`
6. `(0.0, 0.0) == (0.0, 0.0)` (замыкание контура)

Контур получается замкнутым и гладким.

## 4. Центр масштабирования

Для контура пламени удобно использовать центр:

```text
C = (0.0, 4.0)
```

Это приблизительная середина по высоте между основанием и верхней вершиной пламени.

Масштабы для построения:

```text
k = 0.5
k = 1
k = 1.5
k = 2
```
