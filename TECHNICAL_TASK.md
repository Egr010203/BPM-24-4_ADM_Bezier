## 9. Формула кубической кривой Безье

Кубическая кривая Безье задаётся четырьмя контрольными точками `P0`, `P1`, `P2`, `P3`:

```text
B(t) = (1 - t)^3 * P0
     + 3(1 - t)^2 * t * P1
     + 3(1 - t) * t^2 * P2
     + t^3 * P3,
где 0 <= t <= 1.
```

Для координат:

```text
x(t) = (1 - t)^3*x0 + 3(1 - t)^2*t*x1 + 3(1 - t)*t^2*x2 + t^3*x3

y(t) = (1 - t)^3*y0 + 3(1 - t)^2*t*y1 + 3(1 - t)*t^2*y2 + t^3*y3
```

## 10. Формула масштабирования

Масштабирование точки `P = (x, y)` относительно центра `C = (xc, yc)` с коэффициентом `k`:

```text
P' = C + k * (P - C)
```

В координатах:

```text
x' = xc + k * (x - xc)
y' = yc + k * (y - yc)
```
