# BPM-24-4_ADM_Bezier

Курсовая работа по алгоритмам дискретной математики на тему **«Кривые Безье»**.

Проект нужен для выполнения практической части заданий **3.1** и **3.2**:

- **3.1** — построить элементарные кривые Безье третьего порядка по формулам.
- **3.2** — натянуть составную кривую на произвольно выбранный контур и получить изображения этого контура в различных масштабах.

## Выбранная технология

Для реализации выбран **Python**.

Причины выбора:

- удобно хранить и обрабатывать массивы точек;
- библиотека `NumPy` подходит для вычислений координат кривой;
- библиотека `Matplotlib` позволяет быстро строить и сохранять графики;
- код получается компактным и понятным для объяснения на защите.

## Выбранный контур для задания 3.2

В качестве произвольного контура выбрано **стилизованное пламя**.

Этот вариант хорошо подходит для задания 3.2, потому что пламя состоит из нескольких плавных участков, которые удобно представить отдельными кубическими сегментами Безье. В отличие, например, от листа, пламя немного сложнее: оно асимметричное, имеет более выразительные боковые изгибы и верхнюю вершину. Зато такой контур выглядит интереснее и наглядно показывает возможности составных кривых.

Базовые сегменты контура:

| Сегмент | Назначение | P0 | P1 | P2 | P3 |
|---|---|---:|---:|---:|---:|
| 1 | Нижняя левая часть | `(0.0, 0.0)` | `(-2.0, -0.2)` | `(-2.8, 2.5)` | `(-1.2, 4.5)` |
| 2 | Левая верхняя часть к вершине | `(-1.2, 4.5)` | `(-1.0, 5.8)` | `(-0.5, 7.0)` | `(0.0, 8.0)` |
| 3 | Вершина и правая верхняя часть | `(0.0, 8.0)` | `(0.8, 6.8)` | `(2.0, 5.5)` | `(1.4, 3.8)` |
| 4 | Правая нижняя часть и замыкание | `(1.4, 3.8)` | `(2.8, 2.0)` | `(2.2, 0.2)` | `(0.0, 0.0)` |

Смысл точек:

- `P0` и `P3` — опорные точки, через них проходит кривая;
- `P1` и `P2` — управляющие точки, они задают форму изгиба;
- конец каждого сегмента совпадает с началом следующего, поэтому контур получается без разрывов;
- последний сегмент возвращает кривую в начальную точку, поэтому контур замкнут.

## Минимальная логика программы

1. Задать контрольные точки для элементарной кубической кривой.
2. Построить элементарную кривую Безье.
3. Задать список сегментов для контура пламени.
4. Построить каждый сегмент как кубическую кривую Безье.
5. Объединить сегменты в один общий массив точек.
6. Найти центр масштабирования.
7. Построить контур при масштабах `0.5`, `1`, `1.5`, `2`.
8. Сохранить графики в папку `images/`.
